# SL1C3D-L4BS — Role artifacts (commercial-grade)

These artifacts are *required outputs* of the multi-agent loop. Chat is optional; artifacts are mandatory.

## PM
- `Plan.md`: scope, milestones, acceptance criteria, risk register
- `TaskBoard.md`: queue, owners, definition of done

## Researcher
- `ResearchBrief.md`: benchmarks, differentiators, sources
- `DecisionOptions.md`: options, tradeoffs, recommendation

## Engineer / Architect
- `Design.md`: interfaces, boundaries, migration
- `ChangeList.md`: files to touch + why
- `TestPlan.md`: how to validate

## Ops
- `Runbook.md`: start/stop/restart, logs, recovery
- `HealthChecks.md`: commands + expected outputs

## Designer
- `UIContract.md`: tokens, components, states, accessibility
- `Spec.md`: behavior-by-surface

## Security
- `ThreatModel.md`: assets, attack surfaces, controls, audit checks
- `SecretBoundary.md`: what may/never may be stored in dotfiles

## QA
- `GoldenTasks.md`: regression suite definitions
- `ReleaseChecklist.md`: pre-release gating
