# SL1C3D-L4BS — Fullstack scaffolds

These scaffolds are opinionated starter kits designed to align with:
- **Cursor** defaults (font/theme/icons)
- **OpenClaw** workspace protocol (roles + artifacts)
- **Golden suite** gates (validate + secret audit)

## Included
- `web-next/`: Next.js app skeleton
- `api-fastapi/`: FastAPI service skeleton
- `infra-terraform/`: Terraform skeleton
- `obs-otel/`: OpenTelemetry collector skeleton

## Create a new project

Use `~/scripts/new-sl1c3d-project.sh` (generated by this stack) to instantiate a scaffold into a target directory.
