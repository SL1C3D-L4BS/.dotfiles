# Firefox user.js (SL1C3D-L4BS)

**Master template:** `user.js` here is the Betterfox + SL1C3D-L4BS overrides. Copy into each Firefox profile’s directory when updating:

- Profile dirs live under `~/.config/mozilla/firefox/<profile>/`
- Copy: `cp ~/assets/firefox/user.js ~/.config/mozilla/firefox/<profile>/user.js`
- See `~/.config/mozilla/firefox/README-SL1C3D-L4BS.md` for per-profile chrome and update steps.
